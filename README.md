# Brain-games

Five simple brain-games.If give three correct answers you will win.
One wrong answer your lose and will start from first attempt.

## Setup:

Using Makefile:

```bash
$ make install
$ make build
$ make package-install
```

## Without instalation

```bash
$ python3 -m brain_games.scripts.brain_games
```
